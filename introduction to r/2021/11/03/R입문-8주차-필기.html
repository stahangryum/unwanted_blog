데이터 타입

수치형(numeric)

  • 정수, 실수, 지수 등을 나타내며 실수가 기본형
a=345
mode(a)

[1] "numeric"
a=34.5556
mode(a)

[1] "numeric"

논리형(logical)

  • 참(TRUE)과 거짓(FALSE)를 나타낸다.
A = TRUE
mode(A)

[1] "logical"
A = F
mode(A)

[1] "logical"
A = T
mode(A)

[1] "logical"

문자형(character) 

A = "friendship"
mode(A)

[1] "character"
A = "TRUE"
mode(A)

[1] "character"
A = "1"
mode(A)

[1] "character"

복소수형(complex) 

a = 1+3i
b = 3+7i
a+b

[1] 4+10i
mode(a)

[1] "complex"
mode(a+b)

[1] "complex"

데이터 타입 변환

첫 번째 방법 

a = "1"
mode(a)

[1] "character"
mode(a) <- "numeric"
mode(a)

[1] "numeric"
a

[1] 1

두 번째 방법 

a = T
as.numeric(a)

[1] 1
a = F
as.numeric(a)

[1] 0
a = -1
as.logical(a)

[1] TRUE
a = 2
as.logical(a)

[1] TRUE
a = 2.22
as.logical(a)

[1] TRUE
a = 0
as.logical(a)

[1] FALSE
  • 강제형변환
a = "asdfasfasf"
a = as.numeric(a)

Warning message in eval(expr, envir, enclos):
"강제형변환에 의해 생성된 NA 입니다"

mode(a)

[1] "numeric"
as.logical(a)

[1] NA
a = as.logical(a)
mode(a)

[1] "logical"
as.character(a)

[1] NA

서로 다른 형으로 이루어진 벡터의 mode

  • 문자형 섞여있으면 모두 문자형으로 바뀐다.
a <- c(1, T, "1")
a

[1] "1"    "TRUE" "1"   
mode(a)

[1] "numeric"
  • 문자형이 없으면 모두 numeric형으로 바뀐다.
a <- c(1, T)
a

[1] 1 1
mode(a)

[1] "numeric"
  • 서로 다른 형을 가진 벡터를 만들면 한가지 형으로 자동 형 변환이 된다.
  • mode가 두 개일 수 없다.
