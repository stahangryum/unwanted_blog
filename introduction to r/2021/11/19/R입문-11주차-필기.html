Problem

$$\text{Find}\int_0^2 |x^2(x-1)|dx$$

Solution I

$$ \begin{align} \int_0^2 |x^2(x-1)|dx &= \int_0^1 |x^2(x-1)|dx + \int_1^2 |x^2(x-1)|dx\\&=-\int_0^1 x^2(x-1)dx + \int_1^2 x^2(x-1)dx\\&=-\int_0^1 (x^3-x^2)dx + \int_1^2 (x^3-x^2)dx\\&= -\left[\dfrac{1}{4}x^4 - \dfrac{1}{3}x^3\right]_0^1 + \left[\dfrac{1}{4}x^4 - \dfrac{1}{3}x^3\right]_1^2\\&= -\left(\dfrac{1}{4}-\dfrac{1}{3}\right) + \left(\dfrac{16}{4}-\dfrac{8}{3}\right)-\left(\dfrac{1}{4}-\dfrac{1}{3}\right)\\ &=\dfrac{3}{2}\end{align}$$

Solution II 

f = function(x) abs(x**2 *(x-1))

x = seq(0,2, by=0.001)

plot(x, f(x), type = 'l')

xx = runif(10000) * 2
yy = runif(10000) * 4

plot(xx, yy, col = 'gray')
lines(x, f(x), col = 'red')
  • 빨간색 그래프 아래에 있는 넓이가 적분값이다. 
test = function(xx, yy){
    yy < f(xx)
}

A = test(xx, yy)

plot(xx, yy, col = "gray")
lines(x, f(x), col = "red")
points(xx[A], yy[A], col= "red")

$ \text{적분값} = \text{사각형의 크기}\times\dfrac{\text{빨간색 점의 개수}}{\text{전체 점의 개수}}$
