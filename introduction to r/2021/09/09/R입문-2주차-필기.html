함수를 사용하는 방법

- 기본 구조 

함수 이름 <- function(입력){
    함수의 기능
    return 출력
    }

- 예시 

f <- function(d){
    y=2^(-0.25*d)
    return(y)
}

f(8)
0.25

- return을 생략해도 괜찮다. 

g <- function(d){
    y=2^(-0.25*d)
    y
}

g(8)
0.25

벡터의 원소를 뽑는 방법 

x <- c(1,2,-2,4,5,-5,6)
print(x)

[1]  1  2 -2  4  5 -5  6

원소의 번호를 불러서 뽑는 방법

- 첫번째 원소를 뽑고싶다.

파이썬과 다르게 인덱싱은 1부터 시작한다.

print(x[1])

[1] 1

- 두번째 원소를 뽑고싶다. 

print(x[3])

[1] -2

- 첫번째부터 세번째까지의 원소를 뽑고싶다. 

print(x[1:3])

[1]  1  2 -2

print(x[c(1,2,3)])

[1]  1  2 -2

- 2, 5, 6 번째의 원소를 뽑고싶다. 

print(x[c(2, 5, 6)])

[1]  2  5 -5

- 3번째 원소를 제외하고 뽑고싶다. 

print(x)

[1]  1  2 -2  4  5 -5  6

print(x[-3])

[1]  1  2  4  5 -5  6

- 3, 4번째의 원소를 제외하고 뽑고싶다. 

print(x[-c(3,4)])

[1]  1  2  5 -5  6

print(x[-(3:4)])

[1]  1  2  5 -5  6

어떠한 조건을 주고, 그 조건에 만족하는 원소 (혹은 만족하지 않는 원소)를 뽑는 방법

- 양수인 원소를 뽑고싶다. 

print(x)

[1]  1  2 -2  4  5 -5  6

(1) 양수인지 스스로 판단

(2) 양수라면 나간다.

print(x>0)

[1]  TRUE  TRUE FALSE  TRUE  TRUE FALSE  TRUE

x>0의 결과는 TRUE, FALSE가 포함된 벡터

print(x[x>0])

[1] 1 2 4 5 6

- 3이상인 원소를 뽑고싶다. 

print(x>3)

[1] FALSE FALSE FALSE  TRUE  TRUE FALSE  TRUE

print(x[x>3])

[1] 4 5 6

- TRUE 들여다보기 

print(x)

[1]  1  2 -2  4  5 -5  6

array_ = c(TRUE, FALSE, TRUE, FALSE, FALSE, FALSE, FALSE)

print(array_)

[1]  TRUE FALSE  TRUE FALSE FALSE FALSE FALSE

print(x[array_])

[1]  1 -2

- 값이 -2인 원소를 뽑고싶다. 

print(x)

[1]  1  2 -2  4  5 -5  6

print(x[x == -2])

[1] -2

- 응용예제 1 

x <- c(1,2,3,4,5)
y <- x^2

print(x)

  • $y=9$가 되도록 하는 $x$를 구하고싶다.
print(x[y == 9])

[1] 3

    주석 이용 방법

    - 주석은 실행하지 않는 문장이다. 

    a <- 1

print(a)

    a <- 1
#a <- a+1

    - 왜 쓰는가?

    (1) 코드를 나눠서 테스트 할 때

    (2) 코드에 대한 설명이나 메모를 쓰고 싶을 때

    - 과제

    plot(x,y)와 plot(x,y,type='l')을 비교해보기
