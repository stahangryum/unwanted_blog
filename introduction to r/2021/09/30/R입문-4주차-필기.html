조건문의 기본 구조 

if (조건1){
    (조건1이 TRUE이면 실행, FALSE이면 pass)
}
else if (조건2){
    (조건1이 FALSE이고 조건2가 TRUE일 때 실행,)
}
else {
    (조건1과 조건2가 모두 FALSE일 때 실행)

Example 

a = 10

if (a > 0){
    print("a는 양수이다.")
} else if (a < 0){
    print("a는 음수이다.")
} else if (a == 0){
    print("a는 0이다.")  
} else {
    print("a는 실 수가 아니다.")
}

[1] "a는 양수이다."

NOTE2

  • 아래는 잘못된 예이다.
if (5 > 3){
    print("우징엉")
}
else{
    print("오징엉")
}

Error in parse(text = x, srcfile = src): <text>:4:1: 예기치 않은 'else'입니다
3: }

  • 위 코드에서, LINE3에서 컴퓨터는 조건문이 완전히 종료된 것으로 인식하였다.
  • 그런데 갑자기 else를 마주하자 오류를 낸 것.
  • 아래는 옳은 예이다.
if (5 > 3) {
    print("우징엉")
} else {
    print("오징엉")
}

[1] "우징엉"
  • 위 코드처럼 코드가 끝나지 않았음을 컴퓨터에게 알려줘야 한다.

과제

  • if문을 활용하여 제곱근을 계산하는 함수를 만드시오.
mysqrtfun <- function(x){
    if (x >= 0){
        print(sqrt(x))        
    }else{
        print("x는 양수이어야 합니다.")
        }
    }

mysqrtfun(-2)

[1] "x는 양수이어야 합니다."

mysqrtfun(2)

[1] 1.414214
