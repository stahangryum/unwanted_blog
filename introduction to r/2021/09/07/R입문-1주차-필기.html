Solution I

$I_d = I_0 2^{-0.25d}$

$\cfrac{I_d}{I_0} = 0.25 = 2^{-2} = 2^{-0.25d}$

$-2 = -0.25d$

$d = 8m$

Solution II

$d$를 입력하면 $\cfrac{I_d}{I_0}$가 계산되는 아래와 같은 함수를 만들고 싶다.

$\cfrac{I_d}{I_0}=y=f(d)=2^{-0.25d}$

f <- function(d){
    y=2^(-0.25*d)
    y
}

Solution III

$y=f(d)$의 그래프를 구하고 싶다.

d <- seq(from = 0, to = 10, by = 0.01)
y <- 2^(-0.25*d)
plot(d, y, type = "l", lwd = 5)
abline(h = 0.25, col=2, lwd=5, lty = 1)
#col = 색, lwd = 굵기, lty = 점선
#h = 수평선, v = 수직선
